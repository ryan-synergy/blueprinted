# Blueprinted

Drop a Savant RacePoint Blueprint config (`.rpmConfig` zip or folder) and get a
retrofit takeoff: a [SiteWalk](https://ryan-synergy.github.io/sitewalk/)-importable
survey with every room pre-built and flagged, plus equipment / zone-source / signal
reference sheets.

Single file, no dependencies, runs entirely in the browser — configs never upload.

Companion converter script (same logic, Python): kept in the private workspace.
